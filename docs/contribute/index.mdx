## What is the Contribute Guide?

The Contribute Guide contains information about how you can contribute to Passbolt in different ways:

### For Translators
- [Translation](/contribute/translator/translate): Help translate Passbolt into your language
- [Proofreading](/contribute/translator/proofread): Help review and approve translations
- [Externalizing Strings](/contribute/translator/externalize): Help make strings available for translation

### For Developers
- [Feature Requests](/contribute/developer/feature-request): Propose new features
- [Bug Reports](/contribute/developer/bug-report): Report issues you've found
- [Setting up Environment](/contribute/developer/setup): Get started with development

### For Security Researchers
- [Vulnerability Reports](/contribute/security/vulnerability): Report security vulnerabilities
- [Bug Bounty Program](/contribute/security/bug-bounty): Learn about our bug bounty program

## How to get started?

The best way to get started is to:

1. Read our [Code of Conduct](https://www.passbolt.com/code_of_conduct)
2. Join our [Community Forum](https://community.passbolt.com)
3. Choose an area you'd like to contribute to
4. Start contributing!

## Need help?

If you need help getting started or have any questions, you can:

- Ask on our [Community Forum](https://community.passbolt.com)
- Join our [Discord Server](https://discord.gg/passbolt)
- Contact us at [contact@passbolt.com](mailto:contact@passbolt.com)
