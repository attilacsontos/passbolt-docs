These rules should be considered in a firewalled environment:

## Inbound Rules

| Purpose | Protocol Name | Port Number | Transport Layer Protocol |
| ------- | ------------- | ----------- | ------------------------ |
| <span style={{wordBreak: "keep-all"}}>Used for redirecting HTTP traffic to HTTPS for secure communication</span> | HTTP | 80 | TCP |
| <span style={{wordBreak: "keep-all"}}>Enables secure connections to the Passbolt server using SSL/TLS</span> | HTTPS | 443 | TCP |

## Outbound Rules

| Purpose | Protocol Name | Port Number | Transport Layer Protocol |
| ------- | ------------- | ----------- | ------------------------ |
| <span style={{wordBreak: "keep-all"}}>Used for legacy systems or specific cases where HTTPS is not supported</span> | HTTP | 80 | TCP |
| <span style={{wordBreak: "keep-all"}}>Facilitates secure outbound connections for various services, including:</span> <ul><li><span style={{wordBreak: "keep-all"}}>Passbolt's online resources:</span> <ul><li><span style={{wordBreak: "keep-all"}}>`fullBaseUrl` for self-referencing healthcheck</span></li><li><span style={{wordBreak: "keep-all"}}>download.passbolt.com</span></li></ul></li><li><span style={{wordBreak: "keep-all"}}>Secure package repositories</span></li><li><span style={{wordBreak: "keep-all"}}>Bitbucket</span></li><li><span style={{wordBreak: "keep-all"}}>Operating system and software updates</span></li></ul> | HTTPS | 443 | TCP |
| <span style={{wordBreak: "keep-all"}}>Connects to Unique DUO authentication endpoint</span> | HTTPS | 443 | TCP |
| <span style={{wordBreak: "keep-all"}}>Connects to Yubico API endpoints:</span> <ul><li><span style={{wordBreak: "keep-all"}}>api.yubico.com</span></li><li><span style={{wordBreak: "keep-all"}}>api2.yubico.com</span></li><li><span style={{wordBreak: "keep-all"}}>api3.yubico.com</span></li><li><span style={{wordBreak: "keep-all"}}>api4.yubico.com</span></li><li><span style={{wordBreak: "keep-all"}}>api5.yubico.com</span></li></ul> | HTTPS | 443 | TCP |

| <span style={{wordBreak: "keep-all"}}>Sends email notifications through your configured SMTP server</span> | SMTP | Usually 587 | TCP |
