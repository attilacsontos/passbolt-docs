## What is the Development Guide?

The documentation for Passbolt contains several main sections, all accessible from the top navigation bar:
- The [Admin Guide](/admin/) contains all the information for users with the administrator role, e.g. all about the user and admin workspaces.
- The [Hosting Guide](/hosting/) contains all the technical information related to the setup and update of your passbolt self-hosted server.
- The [User Guide](/user/) contains all about the functionalities available to every users.
- The [Development Guide](/development/) that you're currently reading contains all about the technical information related to the backend API and its usages.

If you are new to passbolt you can get some introduction in the next section, otherwise feel free to peruse and find
 what you are looking for using the navigation on the left.
