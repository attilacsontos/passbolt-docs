:::warning
Keeper's default key derivation function is argon2d which is not supported, in order to be able to import your data, a few manual iterations are needed.
:::

## Prerequisites
- KeePass2.X

## Steps to update the file
- Open KeePass2
- Click on *File > Database Settings*

<Figure
  src={
    '/img/import/keeper/keepass_software.png'
  }
  caption="fig. Database Settings Menu"
  alt="Database Settings Menu"
  size={{width: 640}}
/>

- Go the *Security* tab
- Change the "Key derivation function* to **AES-KDF**

<Figure
  src={
    '/img/import/keeper/keepass_algorithm.png'
  }
  caption="fig. Key derivation function update to AES-KDF"
  alt="Key derivation function update to AES-KDF"
  size={{width: 640}}
/>

- Click "OK"
- Save the file
